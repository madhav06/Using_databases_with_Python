Object Oriented Programming Quiz:

1. Which came first, the instance or the class? - class

2. In OOP, what is another name for the 'attributes' of an object? - fields

3. At the moment of creation of a new object, 
Python looks at the _________ definition to define the structure and 
capabilities of the newly created object. - class

4. Which is NOT a good synonym for 'class' in Python? - direction

5. What does this python statement does, if PartyAnimal() is a class?
zap = PartyAnimal(). 
Ans - Use the PartyAnimal template to make a new object and assign it to zap.

6. What is the syntax to look up the fullname attributes in an object 
stored in the variable colleen? - colleen.fullname

7. Which of these statements is used to indicate that class A will
inherit all features of class B? - class A(B).

8. What keyword is used to indicate the start of a method in a python class? - class.


9. What is "self" typically used for in Python method within a class?
Answer - To refer to the instance in which the method is being called. 


10. What does python dir() function show when we pass an object into 
it as a parameter? - It shows methods and attributes of the object.


11. Which of the following is rerely used in OOP? - Destructor.
